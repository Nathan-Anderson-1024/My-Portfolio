Jammming

This project is based off the Spotify API and was developed using React class components. This project focused on integrating the spotify API and uses Implicit Grant Flow to authorize the app to access a user's Spotify data. A user can search for songs by artist, song name, or album and then add songs of their choosing to a new playlist. They can then name this playlist and save the playlist to their spotify account. A user can also fetch all of their public playlists and display their names. Unfortunately due to the limitations of the Spotify API I must add your email associated with your spotify account for the authorization to work. This is because the app is registered as a development app under spotify's terms of service.
